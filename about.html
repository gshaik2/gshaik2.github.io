About This Website

This website was created as part of an educational project to explore the development of professional websites using GitHub Pages. The project focuses on the importance of styling, navigation, and ethical considerations in IT.

Goals of the Project

  • To learn the fundamentals of web development and design.
  • To understand the ethical responsibilities of IT professionals.
  • To demonstrate the ability to create a user-friendly and professional website.

Technologies Used

  • HTML5 for structuring the content.
  • CSS3 for styling and layout.
  • GitHub Pages for hosting the website.

About the Creator

This project was developed by Gufraan Shaik, a student passionate about technology and its ethical application in society. For more information, feel free to explore the rest of the site!
